## 🎨 設計系統

### **色彩方案：**
- **主色調** - 白色 (`#ffffff`)
- **文字色** - 黑色 (`#000000`)
- **強調色** - 橘色 (`#ff6b35`)
- **邊框色** - 淺灰 (`#e0e0e0`)

### **字體系統：**
- **字體家族** - Roboto, Helvetica, Arial
- **字重** - Bold (700) 用於標題
- **響應式** - 根據螢幕大小調整字體大小

### **間距系統：**
- **組件間距** - 使用 Material-UI 的 spacing 系統
- **內邊距** - 統一的 padding 值
- **外邊距** - 一致的 margin 設定
